# 🏦 Bank Management System (Python OOP)

## 📌 Description

A simple menu-driven banking system built using Python.
This project demonstrates Object-Oriented Programming concepts like classes, methods, and encapsulation.

## 🚀 Features

* Deposit money
* Withdraw money
* Check balance
* Menu-driven user interaction

## 🛠️ Concepts Used

* Python OOP (Classes & Objects)
* Loops and Conditionals
* User Input Handling

## ▶️ How to Run

```bash
python bank_system.py
```

* Add file storage for data persistence
